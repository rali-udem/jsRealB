Types de phrases en - jsRealB - Sentence Types

Entrer une structure de phrase selon la langue désirée ou choisir une phrase dans la liste. En décochant des cases de types, on limite le nombre de phrases générées. Les deux dernières cases peuvent prendre une valeur intermédiaire qui n'affiche qu'une possibilité, la première.

Enter a sentence structure in the chosen language or select a sentence from the list. Unchecking checkboxes limits the number of generated sentences. The last two checkboxes can also take an intermediary state that only shows one possibility (the first one).
