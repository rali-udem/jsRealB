Construction of a building

It involves building on an uneven slope an industrial building comprising a metal frame on foundations with well and stringers, paved floor, walls in coated agglomerates, corrugated sheets and corrugated polyester with insulation, heating by suspended air heaters, luminescent lighting.

Adapted from a French example in : A. Gal, G. Lapalme, P. Saint-Dizier, Prolog pour l'analyse automatique du langage naturel, Eyrolles, 1989, pp. 122-129 which explains how to generate a French text using Prolog DCGs.

Input data

The table on the left show the input data: for each task are specified: an id, its length in days and tasks that depend on it. The starting date of the project can be changed here: . .

The system computes the critical path and the earliest and latest dates to which the tasks can take place so that the final project is not delayed. It produces then a text (realized by jsRealB). Hovering the mouse on a task in the graphic highlights it in yellow and displays a tooltip showing informations about this task which is displayed in blue in the generated text and in the table. The immediately preceding and following task are displayed with a red border.
