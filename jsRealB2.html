This is single page loads all jsRealB source files including already some tests in jsRealB2.js.

These tests are checked against the expected value. When the realization is not the expected one, the expression, the output and what was expected are displayed. The number of differences is also displayed.

A warning of a bad value for an option (in French) is also realized, to exercise the warning system.

jsRealB documentation

It is useful for developing using the Javascript console that is displayed using the following key strokes.

  • Chrome : ctrl+shift+j (macOS: cmd+opt+j)
  • Firefox : ctrl+shift+k (macOS: cmd+opt+k) for Firefox
  • Safari : cmd+opt+c
