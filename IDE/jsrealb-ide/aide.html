jsRealB-IDE - Aide

Le but de cette page est d'aider à l'élaboration d'une expression jsRealB et de permettre de faire des démonstrations. Elle permet d'expérimenter en facilitant l'entrée d'une expression Javascript décrivant une structure jsRealB et d'en voir immédiatement la réalisation complète avec l'arbre syntaxique correspondant. On peut également consulter les dictionnaires ainsi que les tables de conjugaison et de déclinaison.

L'écran est divisé verticalement en deux parties. Il est possible de changer la proportion de l'écran occupée par chaque partie en déplaçant la barre qui les sépare.

gauche
Fenêtre contenant un éditeur javascript ACE qui permet d'entrer une expression jsRealB. Si le browser accepte le stockage local, l'expression ainsi que la langue de la dernière génération est conservée entre chaque utilisation de la page.
Rappel des commandes ACE.
Attention: l'arbre et la réalisation affichés à droite correspondent à la dernière expression du code Javascript évaluée avec la fonction prédéfinie eval. On peut utiliser des variables ou des fonctions auxiliaires en autant que la dernière expression retourne une valeur jsRealB. Il est simple de commenter ou dé-commenter une sélection de code avec ctrl-/ (CMD-/ sur Mac) ou placer un bout de code entre /* et */.
droite
Cette partie est elle-même divisée horizontalement en deux parties reconfigurables en déplaçant la barre qui les sépare:
haut
Arbre syntaxique correspondant à l'expression entrée dans la partie gauche de l'écran. En cliquant sur un noeud de l'arbre, on obtient la réalisation du sous-arbre ayant ce noeud comme racine ainsi que la valeur des propriétés internes propagées. Ceci est très utile pour la mise au point. La réalisation de tout l'arbre apparait dans un champ de texte au-dessus de l'arbre.
bas
Consultation des resources linguistiques en tapant une expression régulière dans le champ de texte et en choisissant la langue et le type de ressource dans les menus. Attention cette fenêtre est défilante et il faudra peut-être faire défiler pour voir tous les résultats. Cette région peut aussi être utilisée pour afficher des avertissements de jsRealB au cours de la réalisation.
dictionnaire
trouve toutes les entrées correspondant à l'expression régulière pour lesquelles on affiche la catégorie et la table de conjugaison pour les verbes et de déclinaison pour les autres types d'entrée. En cliquant sur l'entrée de cette table, on indique à la suite l'expression JSON de cette entrée. Ceci est pratique pour créer de nouvelles entrées de dictionnaire semblables à des entrées existantes. En cliquant sur un numéro de table, la table correspondante est affichée.
numéro de conjugaison
montre la table de conjugaison correspondant à un numéro
terminaison de conjugaison
montre la table de conjugaison dont la terminaison correspond à ce qui a été donné dans le champ de texte
numéro de déclinaison
montre la table de déclinaison correspondant à un numéro
terminaison de déclinaison
montre la table de déclinaison dont la terminaison correspond à ce qui a été donné dans le champ de texte
lemmatiser
montre une expression jsRealB permettant de réaliser, si c'est possible, le mot donné en requête qui doit être un seul mot, possiblement une expression régulière
